How Hugo Setup action works

2021-07-15
#servers  #hugo  #github  #deployment 

It’s really easy to host your Hugo site on GitHub, and GitHub Actions for Hugo makes deploy a breeze. And this is how it’s works.

Flow diagram⌗

|files| /archetypes, /content, /static, /themes, config.toml| -+ [branch] main] -> (trigger) Build)
                                                                                    |
                                                                                    *
|files| /assets, /categories, /img, /page, /tags, index.html, 404.html, index.xml, sitemap.xml|
                                                                                        |
                                                                                        +
                                                                                [branch] gh-pages]
                                                                                            |
                                                                                            *
                                                                                    /site.github.io/

Notes⌗

Don’t forget to change your GitHub Pages settings to

  • Branch: gh-pages
  • Directory: / (root)

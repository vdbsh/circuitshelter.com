Configuring DNS settings for Ubuntu with Netplan

2021-08-06
#servers  #linux 

Since 18.04 Ubuntu uses systemd-resolved for name resolution and setting DNS in /etc/resolv.conf will no longer work. Now to set such things we need to make changes to Netplan configuration which delegates name resolution to systemd-resolved.resolv.conf.

vim /etc/netplan/my-dns-config.yaml

Assuming that we are gonna set public Cloudflare DNS for both IPv4 and IPv6 for interface ens3:

# This is my DNS config
network:
  ethernets:
    ens3:
      dhcp4: true
      dhcp4-overrides:
        use-dns: false
      nameservers:
        addresses: [1.1.1.1, 1.0.0.1, 2606:4700:4700::1111, 2606:4700:4700::1001]
  version: 2

Test configuration before apply:

netplan --debug apply

Apply:

netplan apply

Check DNS settings with systemd-resolved:

systemd-resolve --status
